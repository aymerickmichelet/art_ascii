
# ART_ASCII

A cli program to generate ascci art from text

## Run Locally

Clone the project

```bash
  git clone https://github.com/aymerickmichelet/art_ascii.git
```

Go to the project directory

```bash
  cd art_ascii
```

Compile the program

```bash
  ./compiler.sh
```

Run the program

```bash
  ./art_ascii <text>
```
